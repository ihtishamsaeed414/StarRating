Reusbale StarRating component in reactjs

//How to use
import the component in your project 
//the component accepts some props that are given below
1: maxRating(number)
2: color(string)
3: size(number)
4: className(string)
5: messages(array of exact same length as maxRating)
6: defaultRating(number)
7: onSetRating(setter function)

pass the props as for as your use, all the props has also default values
you can pass the props to the component now named <StarRating/>
